# FUTURE_DS_01 - Data Science & Analytics Internship

## Task 1: Business Sales Data Analysis

### Overview
Analyzed Superstore Sales data to identify revenue trends, 
top-selling products, high-value categories, and regional performance.

### Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- GitHub

### Key Findings
- Total Revenue: $2,297,200.86
- Total Profit: $286,397.02
- Total Orders: 5,009
- Top Category: Technology
- Best Region: West
- Loss Making Sub-Categories: Tables, Bookcases
