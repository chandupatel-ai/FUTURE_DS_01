# Task 3 - Marketing Funnel Analysis

## Overview
Analyzed Marketing Funnel data to identify conversion drop-offs,
channel performance, and opportunities to improve
lead-to-customer conversion for a business.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

## Dataset
- Source: Synthetically Generated Dataset
- Rows: 10,000
- Columns: 10

## Key Findings
- Total Visitors: 10,000
- Total Leads: 5,950
- Total Qualified: 2,069
- Total Proposed: 598
- Total Converted: 110
- Overall Conversion Rate: 1.10%
- Total Revenue: $266,548.73
- Avg Revenue per Conversion: $2,423.17
- Best Channel: Referral
- Worst Channel: SEO
- Best Region: North
- Biggest Drop-off: Proposed to Converted (81.6%)

## Files
- `Untitled4.ipynb` - Analysis Notebook
- `Marketing_Funnel_Analysis_Report.pdf` - Final Report

## Insights
- Biggest drop-off is at Proposed to Converted stage (81.6%)
- Referral channel has the highest conversion rate
- Email channel generates the highest revenue
- SEO channel has the lowest conversion rate
- North region outperforms all other regions
- 18-25 age group has the highest conversion rate
