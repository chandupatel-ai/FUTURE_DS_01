# Task 1 - Business Sales Analysis

## Overview
Analyzed Superstore Sales data to identify revenue trends,
top-selling products, high-value categories, and regional performance.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

## Dataset
- Source: Kaggle Superstore Dataset
- Rows: 9,994
- Columns: 21

## Key Findings
- Total Revenue: $2,297,200.86
- Total Profit: $286,397.02
- Total Orders: 5,009
- Total Customers: 793
- Avg Profit Margin: 12.03%
- Top Category: Technology
- Best Region: West
- Loss Making Sub-Categories: Tables, Bookcases

## Files
- `Sample - Superstore.csv` - Raw Dataset
- `Untitled2.ipynb` - Analysis Notebook
- `Superstore_Sales_Analysis_Report.pdf` - Final Report

## Insights
- Technology is the top revenue generating category
- West region leads in total sales performance
- High discounts lead to negative profits
- Sales show consistent growth from 2014 to 2017
